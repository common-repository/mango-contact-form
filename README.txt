== Description ==


Contact Form Plugin , simple and powerfull, using Ajax to submission and request correct fields for a  higth convertion.
Sends to admin email by default, Secure avoid spam.

* Simple design
* A Form of the Right Length
* Effective Error Handling with jquery plugin
* Ajax Submission
* Avoid Spam


== Installation ==

1. Upload the entire mango-contact-form folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
3. Insert this shortcode in the editor [contact-form]
4. Place do_shorcode("[contact-form]") in your code;
